Quick Setup — 3 Steps to Go Live
Get your LinkedIn API token and start the scheduler
Step 01
🔑 Get LinkedIn API Token
Visit linkedin.com/developers → Create App → Request w_member_social permission → Generate access token
Step 02
⚙️ Update .env File
Add your token to the project's .env file as LINKEDIN_ACCESS_TOKEN=your_token
Step 03
🚀 Start the Scheduler
Run the scheduler — it will automatically post at 8:30 AM every day for 15 days
Alternative
📋 Manual Copy-Paste
Use the "Copy" button on each post card and paste directly into LinkedIn. No API needed!
# Install node-cron dependency first:
npm install node-cron

# Preview all posts:
node linkedin_scheduler.js --preview

# Test post today's content immediately:
node linkedin_scheduler.js --now
